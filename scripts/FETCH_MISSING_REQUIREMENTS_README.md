# جلب متطلبات الألعاب الناقصة

## الوصف
هذا السكريبت يبحث عن جميع الألعاب التي لا تحتوي على متطلبات نظام ويحاول جلبها من Steam API تلقائياً.

## كيفية الاستخدام

### الطريقة 1: استخدام ملف Batch (الأسهل)
```bash
# من مجلد backend
.\fetch-missing-requirements.bat
```

### الطريقة 2: استخدام Node.js مباشرة
```bash
# من مجلد backend
node scripts/fetch-missing-requirements.js
```

## ما يفعله السكريبت

1. **يقرأ `games.json`** ويبحث عن الألعاب بدون متطلبات
2. **يتحقق من `fallbackRequirements.json`** لتجنب التكرار
3. **يجلب المتطلبات من Steam** للألعاب الناقصة
4. **يحفظ النتائج** في `fallbackRequirements.json`

## المميزات

- ✅ يعالج 50 لعبة في كل مرة (لتجنب timeout)
- ✅ تأخير 2 ثانية بين كل لعبة (لتجنب rate limiting)
- ✅ يحفظ النتائج تلقائياً
- ✅ يعرض تقرير مفصل عن النتائج

## ملاحظات

- إذا كان لديك أكثر من 50 لعبة ناقصة، شغّل السكريبت عدة مرات
- السكريبت يتخطى الألعاب الموجودة في `fallbackRequirements.json`
- المتطلبات تُحفظ مع `minimum` و `recommended` إذا كانت متوفرة

## الملفات المتأثرة

- `backend/cache/fallbackRequirements.json` - يتم تحديثه بإضافة المتطلبات الجديدة
