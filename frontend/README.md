# Frontend - بتاع العاب

Frontend application built with React + Vite + Tailwind CSS

## 🚀 التشغيل

### تثبيت الحزم
```bash
npm install
```

### وضع التطوير
```bash
npm run dev
```

### بناء المشروع
```bash
npm run build
```

## 📁 البنية

```
frontend/
├── src/
│   ├── components/
│   ├── pages/
│   ├── hooks/
│   ├── services/
│   ├── context/
│   └── ...
├── public/
├── package.json
└── vite.config.js
```

## 🔗 الاتصال بالـ Backend

الـ API URL محدد في `.env`:
```
VITE_API_URL=http://localhost:3001/api
```
